Navbars

Use the navbar helper to display Bootstrap-styled navbars.

Basic navbars

Use navbar to display a navbar, wrapping inside vertical the links that should always be visible and inside horizontal the links that should be collapsed in mobile views.

{% highlight rhtml %} <%= navbar do %> <%= vertical do %> <%= link_to 'Home', root_path %> <% end %> <%= horizontal do %> <%= nav class: 'navbar-right' do %> <%= link_to 'Profile', root_path %> <%= link_to 'Settings', users_path %> <% end %> <% end %> <% end %> {% endhighlight %}

generates a navbar with the following content (try resizing your browser!):

Home
  • Profile
  • Settings
{% highlight html %}
Home
  • Profile
  • Settings
{% endhighlight %}

Inverted colors

Use the :inverted option to invert the palette of colors.

{% highlight rhtml %} <%= navbar inverted: true do %> <%= vertical do %> <%= link_to 'Home', root_path %> <% end %> <% end %> {% endhighlight %}

generates a navbar with a black background:

Home
{% highlight html %}
Home
{% endhighlight %}

Fluid container

Use the :fluid option for a full width navbar, spanning the entire width of your viewport.

{% highlight rhtml %} <%= navbar fluid: true do %> <%= vertical do %> <%= link_to 'Home', root_path %> <% end %> <% end %> {% endhighlight %}

generates a navbar with a fluid container:

Home
{% highlight html %}
Home
{% endhighlight %}

Static

Use the position: :static option to have the navbar scroll away with the page.

{% highlight rhtml %} <%= navbar position: :static do %> <%= vertical do %> <%= link_to 'Home', root_path %> <% end %> <% end %> {% endhighlight %}

generates a navbar that scrolls away with the page:

Home
{% highlight html %}
Home
{% endhighlight %}

Fixed to top

Use the position: :top option to fix the navbar at the top of the page, and the :padding option to specify the padding to leave between the top of the page and the body (defaults to 70px).

{% highlight rhtml %} <%= navbar position: :top do %> <%= vertical do %> <%= link_to 'Home', root_path %> <% end %> <% end %> {% endhighlight %}

generates a navbar fixed to the top with a padding of 70px (click the button to show/hide):

Home
{% highlight html %}
Home
{% endhighlight %}

Fixed to bottom

Use the position: :bottom option to fix the navbar at the bottom of the page, and the :padding option to specify the padding to leave between the body and the bottom of the page (defaults to 70px).

{% highlight rhtml %} <%= navbar position: :bottom, padding: 100 do %> <%= vertical do %> <%= link_to 'Home', root_path %> <% end %> <% end %> {% endhighlight %}

generates a navbar fixed to the bottom with a padding of 100px (click the button to show/hide):

Home
{% highlight html %}
Home
{% endhighlight %}
