Icons

Use the icon helper to display vector icons from different libraries.

Basic example

Use icon without options to display any of the 200 icons available in Bootstrap.

{% highlight rhtml %} <%= icon :user %> {% endhighlight %}

displays the User icon from the Glyphicons library:

{% highlight html %} {% endhighlight %}

Font Awesome example

Use icon with the library: :font_awesome option to display any of the 479 icons available in Font Awesome.

Note: you must load Font Awesome CSS in order to see the icons. See the font_awesome_css helper for details.

{% highlight rhtml %} <%= icon :user, library: :font_awesome %> {% endhighlight %}

displays the User icon from the Font Awesome library:

{% highlight html %} {% endhighlight %}
