Alerts

Use the alert_box helper to display Bootstrap-styled alert messages.

Basic alerts

Use alert_box without options to display a basic informational message.

{% highlight rhtml %} <%= alert_box 'You accepted the Terms of service.' %> {% endhighlight %}

generates the following component (and HTML code):

You accepted the Terms of service.
{% highlight html %}
You accepted the Terms of service.
{% endhighlight %}

Dismissible alerts

Add dismissible: true to add a close button to the alert box.

{% highlight rhtml %} <%= alert_box 'You accepted the Terms of service.', dismissible: true %> {% endhighlight %}

generates the following component (and HTML code):

You accepted the Terms of service.
{% highlight html %}
You accepted the Terms of service.
{% endhighlight %}

Contextual alerts

Set the context option to change the color (and semantic context) of the alert message.

Available contexts are :success, :info (default), :warning and :danger.

{% highlight rhtml %} <%= alert_box 'You accepted the Terms of service.', context: :success %> {% endhighlight %}

displays a success alert message:

You accepted the Terms of service.
{% highlight html %}
You accepted the Terms of service.
{% endhighlight %}

Alerts with HTML content

Pass the alert content to alert_box as a block if you need to include HTML tags or a long text.

{% highlight rhtml %} <%= alert_box context: :warning, dismissible: true do %> Thanks! You accepted the <%= link_to 'Terms of service', '/terms' %>. <% end %> {% endhighlight %}

displays a dismissible warning alert message which includes appropriately styled links and highlighted text:

Thanks! You accepted the Terms of service.
{% highlight html %}
Thanks! You accepted the Terms of service.
{% endhighlight %}
