Panels

Use the panel helper to display Bootstrap-styled panels.

Basic panels

Use panel without options to display a basic panel.

{% highlight rhtml %} <%= panel 'You accepted the Terms of service.' %> {% endhighlight %}

generates the following HTML:

You accepted the Terms of service.
{% highlight html %}
You accepted the Terms of service.
{% endhighlight %}

Panels with heading

Use the :heading options to display a heading above the panel.

{% highlight rhtml %} <%= panel 'You accepted the Terms of service.', heading: 'Congratulations' %> {% endhighlight %}

generates the following HTML:

Congratulations
You accepted the Terms of service.
{% highlight html %}
Congratulations
You accepted the Terms of service.
{% endhighlight %}

Panels with title

Use the :title options to display a title above the panel.

{% highlight rhtml %} <%= panel 'You accepted the Terms of service.', title: 'Congratulations' %> {% endhighlight %}

generates the following HTML:

Congratulations

You accepted the Terms of service.
{% highlight html %}

Congratulations

You accepted the Terms of service.
{% endhighlight %}

Contextual panels

Set the context option to change the color (and semantic context) of the panel.

Available contexts are :default (default), :primary, :success, :info, :warning and :danger.

{% highlight rhtml %} <%= panel 'You accepted the Terms of service.', title: 'Thanks', context: :success %> {% endhighlight %}

displays a success panel message:

Thanks

You accepted the Terms of service.
{% highlight html %}

Thanks

You accepted the Terms of service.
{% endhighlight %}

Panels with custom tag

Set the tag option to use a different HTML tag to wrap the panel.

{% highlight rhtml %} <%= panel 'You accepted the Terms of service.', tag: :aside %> {% endhighlight %}

displays a panel wrapped in an <aside> tag:

You accepted the Terms of service.
{% highlight html %}
You accepted the Terms of service.
{% endhighlight %}
