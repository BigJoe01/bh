Dropdowns

Use the dropdown helper to display Bootstrap-styled dropdowns.

Basic dropdown

Use dropdown without options to display a button with the given caption and the included links in a dropdown menu.

{% highlight rhtml %} <%= dropdown 'Menu' do %> <%= link_to 'Home', root_path %> <%= link_to 'Users', users_path %> <%= link_to 'Profile', profile_path %> <% end %> {% endhighlight %}

generates the following component (and HTML code):

  • Home
  • Users
  • Profile
{% highlight html %}
  • Home
  • Users
  • Profile
{% endhighlight %}

Split dropdown button

Use dropdown with the split: true option to show a split button that only toggles the dropdown when clicked on the right side.

{% highlight rhtml %} <%= dropdown 'Menu', split: true do %> <%= link_to 'Home', root_path %> <%= link_to 'Users', users_path %> <%= link_to 'Profile', profile_path %> <% end %> {% endhighlight %}

generates the following component (and HTML code):

  • Home
  • Users
  • Profile
{% highlight html %}
  • Home
  • Users
  • Profile
{% endhighlight %}

Dropup button

Use dropdown with the direction: :up option to show a dropup, that is, a menu that appears above the button.

{% highlight rhtml %} <%= dropdown 'Menu', direction: :up do %> <%= link_to 'Home', root_path %> <%= link_to 'Users', users_path %> <%= link_to 'Profile', profile_path %> <% end %> {% endhighlight %}

generates the following component (and HTML code):

  • Home
  • Users
  • Profile
{% highlight html %}
  • Home
  • Users
  • Profile
{% endhighlight %}

Grouped dropdowns

Use dropdown with the groupable: true option to have multiple dropdowns grouped on the same line, rather than stacked.

{% highlight rhtml %} <%= dropdown 'Menu', groupable: true do %> <%= link_to 'Home', root_path %> <%= link_to 'Users', users_path %> <% end %> <%= dropdown 'Profile', groupable: true do %> <%= link_to 'Edit Profile', edit_profile_path %> <% end %> {% endhighlight %}

generates the following component (and HTML code):

  • Home
  • Users
  • Edit Profile
{% highlight html %}
  • Home
  • Users
  • Edit Profile
{% endhighlight %}

Contextual dropdown

Use dropdown with the :context option to change the color (and semantic context) of the toggle button.

Available contexts are :default (default), :primary, :success, :info, :warning and :danger.

{% highlight rhtml %} <%= dropdown 'Menu', context: :info do %> <%= link_to 'Home', root_path %> <%= link_to 'Users', users_path %> <% end %> {% endhighlight %}

displays an info (blue background) button to toggle a dropdown:

  • Home
  • Users
{% highlight html %}
  • Home
  • Users
{% endhighlight %}

Custom button size

{% highlight rhtml %} <%= dropdown 'Menu', size: :extra_small do %> <%= link_to 'Home', root_path %> <%= link_to 'Users', users_path %> <% end %> {% endhighlight %}

displays an extra small button to toggle a dropdown:
