Modals

Use the modal helper to display Bootstrap-styled modals.

Basic example

Use modal with the :body option to display a button which toggles a modal with the given text when clicked.

{% highlight rhtml %} <%= modal body: 'Do what you want!' %> {% endhighlight %}

generates the following component (and HTML code):

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

Sizes

Use modal with the :size option to display either a large or a small modal.

{% highlight rhtml %} <%= modal body: 'Do what you want!', size: 'small' %> {% endhighlight %}

generates a button that toggles a small modal:

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

Contexts

Use modal with the button: :context option to change the color (and semantic context) of the toggle button.

Available contexts are :default (default), :primary, :success, :info, :warning and :danger.

{% highlight rhtml %} <%= modal title: 'Terms of service', body: 'Do what you want!', button: {context: :info} %> {% endhighlight %}

displays an info (blue background) button to toggle a modal:

Terms of service

Do what you want!
{% highlight html %}

Terms of service

Do what you want!
{% endhighlight %}

With title

Use modal with the :title option to display a custom title at the top of the modal.

{% highlight rhtml %} <%= modal title: 'Terms of service', body: 'Do what you want!' %> {% endhighlight %}

generates the following component (and HTML code):

Terms of service

Do what you want!
{% highlight html %}

Terms of service

Do what you want!
{% endhighlight %}

With button caption

Use modal with the {button: :caption} option to display a custom caption on the toggle button.

{% highlight rhtml %} <%= modal button: {caption: 'Click me!'}, body: 'Do what you want!' %> {% endhighlight %}

generates the following component (and HTML code):

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

With button size

Use modal with the {button: :size} option to change the size of the toggle button.

{% highlight rhtml %} <%= modal button: {size: :xs}, body: 'Do what you want!' %> {% endhighlight %}

displays an extra-small button to toggle a modal:

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

With HTML content

Pass the content to modal as a block if you need to include HTML tags or a long text.

{% highlight rhtml %} <%= modal title: 'Terms of service', button: {size: :large, context: :info} do %>
Please accept the Terms of service.
<% end %> {% endhighlight %}

displays a large, info button (blue) with the caption "Continue" that toggles a small modal with a title and HTML content:

Terms of service

Please accept the Terms of service.
{% highlight html %}

Terms of service

Please accept the Terms of service.
{% endhighlight %}
