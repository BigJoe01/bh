Modals

Use the modal helper to display Bootstrap-styled modals.

Basic modals

Use modal without options to display a button which toggles a modal with the given body when clicked.

{% highlight rhtml %} <%= modal 'Do what you want!' %> {% endhighlight %}

generates the following HTML:

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

Custom-sized modals

Use modal with the :size option to display either a large or a small modal.

{% highlight rhtml %} <%= modal 'Do what you want!', size: 'small' %> {% endhighlight %}

generates a button that toggles a small modal:

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

Modals with custom title

Use modal with the :title option to display a custom title at the top of the modal.

{% highlight rhtml %} <%= modal 'Do what you want!', title: 'Terms of service' %> {% endhighlight %}

generates the following HTML:

Terms of service

Do what you want!
{% highlight html %}

Terms of service

Do what you want!
{% endhighlight %}

Modals with custom button caption

Use modal with the {button: :caption} option to display a custom caption on the toggle button.

{% highlight rhtml %} <%= modal 'Do what you want!', button: {caption: 'Click me!'} %> {% endhighlight %}

generates the following HTML:

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

Modals with custom button context

Use modal with the button: :context option to change the color (and semantic context) of the toggle button.

Available contexts are :default (default), :primary, :success, :info, :warning and :danger.

{% highlight rhtml %} <%= modal 'Do what you want!', title: 'Terms of service', button: {context: :info} %> {% endhighlight %}

displays an info (blue background) button to toggle a modal:

Terms of service

Do what you want!
{% highlight html %}

Terms of service

Do what you want!
{% endhighlight %}

Modals with custom button size

Use modal with the {button: :size} option to change the size of the toggle button.

{% highlight rhtml %} <%= modal 'Do what you want!', button: {size: :xs} %> {% endhighlight %}

displays an extra-small button to toggle a modal:

Modal

Do what you want!
{% highlight html %}

Modal

Do what you want!
{% endhighlight %}

Advanced modals

To include HTML tags or a long text in the modal, pass your content in a block.
You can also specify a custom id which will be added to the modal’s <div> tag.

{% highlight rhtml %} <%= modal title: 'Terms of service', size: :large, id: 'modal' do %>
Please accept the Terms of service.
<% end %> {% endhighlight %}

displays a button that toggles a large modal with a title and HTML content:

Terms of service

Please accept the Terms of service.
{% highlight html %}

Terms of service

Please accept the Terms of service.
{% endhighlight %}
