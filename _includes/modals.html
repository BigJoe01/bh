Modals

Use the modal helper to display Bootstrap-styled modals.

Basic modals

Use modal with the :body option to display a button that toggles a model with the body when clicked. For instance:

{% highlight rhtml %} <%= modal title: 'Show terms of service', body: 'Do what you want!' %> {% endhighlight %}

generates the following component (and HTML code):

Show terms of service

Do what you want!
{% highlight html %}

Terms of service

Do what you want!
{% endhighlight %}

Complex modals

Combine multiple options and include an HTML block in the modal as needed. For instance:

{% highlight rhtml %} <%= modal title: 'Terms of service', size: :small, button: {caption: 'Continue', size: :large, context: :info} do %>
Please accept the Terms of service.
<% end %> {% endhighlight %}

displays a large, info button (blue) with the caption "Continue" that toggles a small modal with a title and HTML content:

Terms of service

Please accept the Terms of service.
{% highlight html %}

Terms of service

Please accept the Terms of service.
{% endhighlight %}
