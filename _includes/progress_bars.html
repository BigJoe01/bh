Progress bars

Use the progress_bar helper to display Bootstrap-styled progress bars.

Basic progress bar

Use progress_bar with the :percentage option to show a progress bar.

{% highlight rhtml %} <%= progress_bar percentage: 30 %> {% endhighlight %}

generates the following HTML:

30%
{% highlight html %}
30%
{% endhighlight %}

Progress bar with label

Use progress_bar with the :label option to show a label on the progress bar.

Set :label to a string to use a custom label, or to true to use the value as the label.

{% highlight rhtml %} <%= progress_bar percentage: 30, label: true %> <%= progress_bar percentage: 30, label: "thirty percent" %> {% endhighlight %}

generates the following components (and HTML code):

30%
thirty percent
{% highlight html %}
30%
thirty percent
{% endhighlight %}

Contextual progress bars

Use progress_bar with the :context option to change the color (and semantic context) of the progress bar.

Available contexts are :success, :info, :warning and :danger.

{% highlight rhtml %} <%= progress_bar percentage: 30, context: :warning %> {% endhighlight %}

generates the following components (and HTML code):

30% (warning)
{% highlight html %}
30% (warning)
{% endhighlight %}

Striped progress bars

Use progress_bar with the striped: true option to display a striped progress bar.

{% highlight rhtml %} <%= progress_bar percentage: 30, striped: true %> {% endhighlight %}

generates the following components (and HTML code):

30%
{% highlight html %}
30%
{% endhighlight %}

Animated progress bars

Use progress_bar with the animated: true option to display a striped animated progress bar.

{% highlight rhtml %} <%= progress_bar percentage: 30, animated: true %> {% endhighlight %}

generates the following components (and HTML code):

30%
{% highlight html %}
30%
{% endhighlight %}

Stacked progress bars

Use progress_bar with an array of options (rather than a single Hash) to display a set of stacked progress bars.

{% highlight rhtml %} <%= progress_bar [{percentage: 30, context: :success, label: 'Completed'}, {percentage: 40, context: :warning, animated: true, label: 'Pending'}] %> {% endhighlight %}

generates the following components (and HTML code):

Completed
Pending
{% highlight html %}
Completed
Pending
{% endhighlight %}
