Glyphicons

Use the glyphicon helper to display any of the 200 glyphicons available in Bootstrap.

Basic example

Use glyphicon with the name of any of the 200 glyphicons available in Bootstrap to display the glyph.

{% highlight rhtml %} <%= glyphicon :ok, title: 'Approved' %> {% endhighlight %}

displays the ok glyphicon with the alternate title "Approved" on hover:
