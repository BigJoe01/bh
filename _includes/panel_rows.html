Panel rows

Use the panel_row helper a row of horizontally-aligned Bootstrap-styled panels.

Basic row of panels

Embed a set of panels with panel_row to align them horizontally in the same row.

{% highlight rhtml %} <%= panel_row column_class: 'col-sm-4' do %> <%= panel body: 'Panel #1' %> <%= panel body: 'Panel #2' %> <%= panel body: 'Panel #3' %> <% end %> {% endhighlight %}

generates the following HTML:

Panel #1
Panel #2
Panel #3
{% highlight html %}
Panel #1
Panel #2
Panel #3
{% endhighlight %}

Complex row of panels

Combine multiple panels with different options HTML content as needed.

{% highlight rhtml %} <%= panel_row column_class: 'col-sm-4' do %> <%= panel title: 'User', context: :info do %>
<%= icon :user %> John Smith
<% end %> <%= panel title: 'Phone' do %>
<%= icon :earphone %> 323-555-5555
<% end %> <%= panel title: 'City' do %>
<%= icon :record %> Los Angeles, CA
<% end %> <% end %> {% endhighlight %}

displays a row of three panels with title and HTML body:

User

John Smith

Phone

323-555-5555

City

Los Angeles, CA
{% highlight html %}

User

John Smith

Phone

323-555-5555

City

Los Angeles, CA
{% endhighlight %}
