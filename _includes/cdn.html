Bootstrap CDN

Use the bootstrap_css, bootstrap_theme_css and bootstrap_js helpers to load the main Bootstrap stylesheets and javascript files from MaxCDN.

Bootstrap CSS file

Use bootstrap_css without options to get the URL of the most up-to-date Bootstrap CSS file. For instance:

{% highlight rhtml %} <%= stylesheet_link_tag bootstrap_css %> {% endhighlight %}

generates the following HTML code:

{% highlight html %} {% endhighlight %}

Older versions

Use the :version option to get the URL of a specific version of Bootstrap CSS. For instance:

{% highlight rhtml %} <%= stylesheet_link_tag(bootstrap_css version: '3.1.0') %> {% endhighlight %}

generates the following HTML code:

{% highlight html %} {% endhighlight %}

Non-minified versions

Use the :minified option to get the non-minified version of Bootstrap CSS. For instance:

{% highlight rhtml %} <%= stylesheet_link_tag(bootstrap_css minified: false) %> {% endhighlight %}

generates the following HTML code:

{% highlight html %} {% endhighlight %}

Fixed-scheme URL

Use the :scheme option to get either the HTTP or HTTPS version of the URL for Bootstrap CSS. For instance:

{% highlight rhtml %} <%= stylesheet_link_tag(bootstrap_css scheme: :http) %> {% endhighlight %}

generates the following HTML code:

{% highlight html %} {% endhighlight %}

Bootstrap Theme CSS

Use bootstrap_theme_css to get the URL of Bootstrap Theme CSS. Accepts the same options as bootstrap_css. For instance:

{% highlight rhtml %} <%= stylesheet_link_tag(bootstrap_theme_css schema: 'https') %> {% endhighlight %}

generates the following HTML code:

{% highlight html %} {% endhighlight %}

Bootstrap JS

Use bootstrap_js to get the URL of the Bootstrap JS file. Accepts the same options as bootstrap_css. For instance:

{% highlight rhtml %} <%= javascript_include_tag bootstrap_js(version: '3.1.0', minified: false) %> {% endhighlight %}

generates the following HTML code:

{% highlight html %} {% endhighlight %}
