Overview

Combine the beauty of Bootstrap’s design and the expressiveness of Ruby to get the best of both worlds with the least amount of keystrokes.

Bootstrap is a great framework, but requires you to write many lines of HTML code even for simple components.
For instance, to show a dismissible alert, you need to write the following lines of HTML:

{% highlight html %}
You accepted the Terms of service.
{% endhighlight %}

Writing this for every dismissible alert in a web site is cumbersome, repetitive, and prone to errors.

Bh solves this problem by providing a set of Ruby helpers. With Bh, you can write the example above in just one line of code:

{% highlight rhtml %} <%= alert_box 'You accepted the Terms of service.', dismissible: true %> {% endhighlight %}

How to install

Bh helpers can be used with Ruby on Rails, and with any other Ruby web framework that supports ERB templates.

Require the bh gem in the project’s Gemfile with gem 'bh', and you are good to go!

Bh does not overwrite any default helper or Bootstrap style, giving you the freedom to use both Bh helpers and the traditional way of writing Bootstrap components.
